# Bookble

**Bookble** is a comprehensive digital library app for book lovers. Explore and read novels and books, listen to audiobooks, and view summaries easily and seamlessly. Bookble combines modern design, dynamic features, and multi-platform support to make reading and listening a delightful experience.

---

## üåü Key Features

* **Read Books & Novels**: Supports EPUB and PDF formats for a smooth and comfortable reading experience, with adjustable font size, themes, and layout options.
* **Audiobooks**: Listen to your favorite books and novels on the go, with playback controls and bookmarks to track progress.
* **Summaries**: Access concise summaries of books and novels to grasp the main ideas quickly and efficiently.
* **Personal Library**: Organize your collection, add favorites, and keep track of reading progress with a well-designed library interface.
* **Multi-language Support**: Switch seamlessly between LTR and RTL interfaces for different languages.
* **Dynamic Book Covers**: Covers adapt dynamically to language and style, with automatic adjustment of text font, color, and placement.

## üöÄ Installation

1. **Clone the repository:**

```bash
git clone https://github.com/username/bookble.git
cd bookble
```

2. **Install dependencies:**

```bash
# For Android
./gradlew build
# For web interface (if applicable)
npm install
```

3. **Run the application:**

* **Android**: Open the project in Android Studio and run it on an emulator or a real device.
* **Web**: `npm start`
